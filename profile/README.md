# 🌍 Our Mission

Baobab Chain Labs is an independent open-source organization building practical blockchain infrastructure for emerging economies.

Inspired by Africa's iconic **Baobab Tree**—a symbol of resilience, community, and longevity—we develop public goods that make financial services, digital identity, agricultural finance, and remittances more accessible through Stellar and Soroban.

While our first solutions are inspired by challenges across Nigeria and Africa, every project is designed with global interoperability and scalability in mind.

# 🚀 Our Projects

| Repository | Description |
|------------|-------------|
| **🌾 [FarmChain Receipts](https://github.com/baobab-chain/farmchain-receipts)** | Tokenized warehouse receipts that enable farmers to use stored commodities as verifiable collateral for financing. |
| **🌦️ [HarvestGuard](https://github.com/baobab-chain/harvestguard)** | Parametric crop insurance powered by Stellar, delivering automated payouts based on weather and climate events. |
| **🎓 [CredVerify](https://github.com/baobab-chain/credverify)** | Verifiable academic and professional credentials secured on Stellar for instant verification anywhere in the world. |
| **💸 [Remit Last Mile](https://github.com/baobab-chain/remit-last-mile)** | Infrastructure connecting Stellar-powered remittances to local cash-out agents for affordable last-mile delivery. |
| **🤝 [Diaspora Circle](https://github.com/baobab-chain/diaspora-circle)** | Cross-border rotating savings (Ajo/Esusu) protocol enabling diaspora communities to save and contribute seamlessly using Stellar assets. |

# 🛠️ What We Believe

We build open infrastructure that is:

- 🌍 **Accessible** — Financial tools should work for everyone, regardless of geography.
- 🔓 **Open Source** — Community-owned protocols create stronger ecosystems.
- ⚡ **Affordable** — Low transaction costs unlock real-world adoption.
- 🧩 **Composable** — Projects should integrate easily with one another.
- 🌱 **Sustainable** — Infrastructure should continue serving communities for years to come.

# 🤝 Contributing

We welcome developers, designers, technical writers, security researchers, and blockchain enthusiasts.

Across our repositories you'll find issues covering:

- 📖 Documentation
- 🐛 Bug fixes
- ✨ New features
- 🧪 Testing

- 🌐 Stellar & Soroban integrations

If you're participating in the **Drips Wave Program**, each issue includes:

- Problem statement
- Implementation scope
- Technical requirements
- Acceptance criteria

making it easy to contribute regardless of experience level.

Before opening a Pull Request:

- Read each repository's `CONTRIBUTING.md`
- Pick an open issue
- Discuss larger design changes before implementation
- Follow the project's coding standards
